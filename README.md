moyiui
======

moyi UI

未来，没有框架，只有库。

按照自己的业务拆分使用，使用gulp可以自动生成sprite图

##常用工具类

@import "moyi/untility";

##网格

@import "moyi/grid";
@import "moyi/grid-tool";

##组件

@import "moyi/button";

##图标

http://fontello.com/进行定制 

@import "moyi/icon";

使用font-awesome字体

@import "fontawesome/font-awesome";
